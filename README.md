# Groundwater Trend Analysis: ET/Precipitation Ratio and Aridity Index

## Purpose

This repository contains the analysis pipeline for examining the relationship between groundwater level trends and the ratio of evapotranspiration (ET) to precipitation across 50 irrigated agricultural well sites in the United States. Sites span six regions and cover the period 2000–2020 (with incomplete records for up to 7 sites).

The central output is a scatter plot with the ET/Precipitation ratio on the x-axis and the 20-year groundwater trend (m/yr) on the y-axis, where each point represents a single well. This analysis is repeated for four spatial bounding box scales: 1 km, 2 km, 4 km, and 10 km. Aridity Index is calculated per well site, and Spearman's rank correlation is applied to the final figure.

## Folder Descriptions

- **`aridity_index/`** — Computes the Aridity Index per well site from the Global Aridity Index raster. Produces Budyko curve visualizations at multiple scales.

- **`cultivation_groundwater/`** — Imports USGS groundwater data and USDA CDL crop data; appends cultivation percentages for each study site.

- **`data/`** — Raw input data (not tracked by Git). See [Data Access](#data-access) for download instructions.
  - `cultivation/` — USDA CDL rasters for 2008–2020, one folder per year (`.tif`, `.tfw`, `.aux`, metadata)
  - `gw_data/` — USGS groundwater site metadata and water level records

- **`data_viz/`** — Produces the final scatter plot figure of ET/Precipitation ratio vs. groundwater trend.

- **`ET/`** — Scripts for extracting and processing evapotranspiration data.
  - `usgs_water_level_cleaning.ipynb` — Cleans and filters raw USGS water level records
  - `open-et-timeseries.ipynb` — Extracts OpenET data via the Polygon Raster API
  - `modis-et-timeseries.ipynb` — Extracts MODIS ET time series
  - `water_level_et_plots.ipynb` — Plots water level and ET together for visual inspection

- **`outputs/`** — CSV outputs written by pipeline scripts. Created automatically on first run.

- **`precipitation/`** — Scripts for downloading and processing CHIRPS precipitation data.
  - `chirps_API.ipynb` — Downloads daily CHIRPS TIFF files via API
  - `chirps_precipt_USGS_analysis_openET.ipynb` — Computes site-level precipitation normals and ET:Precipitation ratios

- **`precipitation_variation/`** — Computes precipitation variability per site from the CHIRPS daily rasters.

- **`scratch/`** — Exploratory and developmental notebooks not part of the main pipeline.

- **`spearman_rank/`** — Applies Spearman's rank correlation to the ET/Precipitation ratio and groundwater trend data.

- **`yml/`** — Conda environment file for reproducing the project environment.

## Data Access

Create a folder named `data/` in the root of the repository. All datasets should be placed within this folder as described below.

### USGS Well Data
Groundwater monitoring site metadata and associated groundwater level records are hosted on HydroShare. Download the CSV and place it in `data/gw_data/`:

[Groundwater Site Data – HydroShare](https://www.hydroshare.org/resource/04b9eebf0d4b4490a15e1f27be46f7e4/)

### USDA Cropland Cultivation Data
Cultivation data are sourced from the USDA Cropland Data Layer (CDL), available for 2008–2020. Download the National CDL zip files for each year and place them in `data/cultivation/`. Once in place, unzip each file.

[Cropland Data Layer – USDA](https://www.nass.usda.gov/Research_and_Science/Cropland/Release/index.php)

### OpenET Evapotranspiration Data
ET data are extracted via the [OpenET Polygon Raster API](https://openet-api.org/#/Retrieve%20Raster%20Data/polygon_raster_timeseries_polygon_post). An account and personal API key are required. See `ET/open-et-timeseries.ipynb` for extraction instructions.

### CHIRPS Precipitation Data
Precipitation data are sourced from the CHIRPS v3.0 dataset (daily satellite-based rasters, 1981–present). No account or API key is required. Run `precipitation/chirps_API.ipynb` to download TIFF files to a `chirps_data/` folder.

### Aridity Index Data
Data were obtained from the [Global Aridity Index and Potential Evapotranspiration (ET₀) Database: Version 3.1](https://doi.org/10.6084/m9.figshare.7504448), available through Figshare. Download and unzip the `Global-AI_ET0_annual_v3_1` folder. This analysis uses the annual raster file `ai_v31_yr.tif`.

## Statistical Methods and Sensitivity Testing

### Spearman's Rank Correlation
Spearman's rank correlation is a non-parametric measure of the monotonic relationship between two variables. It is used here to assess whether wells with a higher ET/Precipitation ratio tend to show stronger declining (or recovering) groundwater trends over the 20-year study period. Because the data are ranked rather than used as raw values, Spearman's is appropriate when the relationship between variables may not be linear or when normality cannot be assumed.

### Sensitivity Testing
The analysis is repeated across four spatial scales — 1 km, 2 km, 4 km, and 10 km bounding boxes — to evaluate how sensitive the ET estimates and resulting correlations are to the spatial extent used when sampling ET around each well.

## Directions for Use

> The following directions pertain to the **1 km bounding box** analysis. All scripts should be run in the order listed. Each step writes outputs to the `outputs/` folder, which is created during step 1 and referenced by all subsequent steps.

### Step 1 — Cultivation

Run `cultivation_groundwater/cult_gwl_1km.ipynb`.

Imports USGS groundwater data and USDA CDL crop data; appends cultivation percentages for each study site.

**Output:** `outputs/gwl_cult_1km.csv`

### Step 2 — Evapotranspiration

Run `ET/open-et-timeseries.ipynb`. Before running, update the following variables:

**Site metadata file path**
```python
clean_site_data = "/capstone/aridgw/data/site_data/clean_site_waterdata_04222026.csv"
```

**API key**
```python
header = {"Authorization": "INSERT API KEY"}
```

**Bounding box side length (km)**
```python
size_km = 1
```

**Date ranges** — Use version `2.1` for 2020 and later, as earlier versions may not accurately represent recent data.
```python
date_ranges = [
    ("2000-01-01", "2009-12-31", 2.0),
    ("2010-01-01", "2019-12-31", 2.0),
    ("2020-01-01", "2025-12-31", 2.1),  # v2.1 required for 2020+
]
```

**Output:** `outputs/gwl_cult_et_1km.csv`

### Step 3 — Precipitation

**3a.** Run `precipitation/chirps_API.ipynb` to download all daily CHIRPS TIFF files.

**Output:** `chirps_data/` folder (~7,671 files)

**3b.** Run `precipitation/chirps_precipt_USGS_analysis_openET.ipynb`. This script uses `outputs/gwl_cult_et_1km.csv` and the CHIRPS TIFFs to compute site-level precipitation normals and ET:Precipitation ratios. It averages daily CHIRPS rasters from 2000–2020 into a single precipitation normal raster, then extracts a mean precipitation value (mm/yr) per site using a 1 km bounding box.

To change the buffer size, edit `buffer_km=0.5` in the following line (use half the desired total buffer width):
```python
def extract_precip_with_buffer(lon, lat, src, buffer_km=0.5):
```

When changing buffer size, also update all related output filenames, variable names, and folder names accordingly (e.g., `1km` → `2km`).

**Outputs:**
- `outputs/chirps_daily_precip_normal_2000_2020.tif`
- `outputs/site_summary_1km.csv`
- `outputs/et_precipt_ratio_1km.csv`

**3c.** Run `precipitation_variation/precipt_variation.ipynb`. Uses `outputs/et_precipt_ratio_1km.csv` and the precipitation normal raster to compute precipitation variability per site.

To change the buffer size, edit the following line:
```python
half_size = 0.5 / 111.32  # 0.5 km → 1 km total buffer
```

**Output:** `outputs/et_precipt_ratio_1km.csv` (updated with `precip_variation` column)

### Step 4 — Final Figure

Run `data_viz/data_visualization.ipynb` to produce the scatter plot of ET/Precipitation ratio vs. groundwater trend (m/yr).

### Step 5 — Aridity Index

Run `aridity_index/aridity_index.ipynb` to compute the Aridity Index per well site and generate Budyko curve visualizations.

### Step 6 — Spearman's Rank Correlation

Run `spearman_rank/spearman_stats.ipynb` to compute Spearman's rank correlation between the ET/Precipitation ratio and groundwater trend across all sites.
